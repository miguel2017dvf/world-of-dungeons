# World of Dungeons

An action RPG dungeon crawler built with Phaser 3, TypeScript, and Vite.

Focused on:
- Fast-paced combat
- Elemental abilities
- Dungeon exploration
- Guardian systems
- Skill progression
- Future multiplayer features

## Tech Stack

- Phaser 3
- TypeScript
- Vite
- Node.js

## Status

Early development / prototype phase.
